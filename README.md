# weather-app

Консольное приложение для получения текущих погодных условий через OpenWeatherMap API.

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

## Описание

Приложение позволяет получить актуальные данные о погоде для любого города через командную строку.


**Основные возможности:**
* Получение температуры, описания погоды, влажности, давления и скорости ветра.
* Поддержка любого города мира (через OpenWeatherMap).
* Отображение данных на русском языке.

* Простой консольный интерфейс.
